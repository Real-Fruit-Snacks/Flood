# Security Policy

## Supported Versions

Only the latest release is supported with security updates.

## Reporting a Vulnerability

Report vulnerabilities through [GitHub Security Advisories](https://github.com/Real-Fruit-Snacks/Flood/security/advisories/new).

- **Acknowledgment:** Within 48 hours
- **Assessment:** Within 7 days
- **Fix timeline:** Within 90 days

## Important Note

Flood is an authorized security testing tool. The following behaviors are **features, not bugs**:

- Sending large volumes of HTTP requests to targets
- Brute-forcing directories, files, and parameters
- Recursive discovery of web application structure
- Proxy integration for traffic interception

> **Authorization Required:** Only use Flood against systems you have explicit written permission to test. Unauthorized use may violate computer fraud laws.
